Usage

integrate_const(stepper, ode_system, y, t0, t1, dt, save_state = FALSE)

Arguments

stepper
A stepper object, created by stepper (other types will be supported soon).
ode_system
The ode_system system, created by ode_system
y
Initial conditions
t0
Time to start the integration
t1
Time to finish the integration
dt
Step size
save_state
Return information about intermediate points as an attribute?
