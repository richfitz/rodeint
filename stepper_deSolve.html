Stepper (deSolve wrappers)

Usage

make_stepper_deSolve(category, type, ...)


make_stepper_deSolve_basic(type)


make_stepper_deSolve_controlled(type, abs_tol = 1e-06, rel_tol = 1e-06)

Arguments

category
Either "basic" or "controlled"
type
The type of stepper (e.g. "runge_kutta4")
...
Additional parameters passed from make_stepper to either make_stepper_basic (none allowed) or make_stepper_controlled.
abs_tol
Absolute tolerance (see atol in deSolve docs for now)
rel_tol
Relative tolerance (see rtol in deSolve docs for now)
