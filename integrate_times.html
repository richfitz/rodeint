Adaptively Integrate an ODE System

Usage

integrate_times(stepper, ode_system, y, times, dt)

Arguments

stepper
A stepper object, created by stepper (other types will be supported soon).
ode_system
The ode_system system, created by ode_system
y
Initial conditions
times
Vector of times. First time is start time, last time is end time.
dt
Initial step size (will be tuned for controlled steppers -- see odeint documentation)

Adaptively Integrate an ODE System

Description

Integrate a system of ODEs at fixed times (perhaps adaptively). This version always saves state -- that is the reason for calling it.
